# ENCompIF-2021-2025: Análise de Pesquisa na Rede Federal (2021-2025)

Uma análise completa das tendências de pesquisa e publicações apresentadas no **ENCompIFs** (Encontro Nacional de Computação dos Institutos Federais) durante o período de 2021 a 2025.

## 📊 Sobre o Projeto

Este projeto investiga as tendências de pesquisa e áreas de interesse dos alunos e professores dos Institutos Federais de Educação, Ciência e Tecnologia através da análise de trabalhos publicados no ENCompIFs, o principal evento de computação da rede federal de educação tecnológica do Brasil.

### 🎯 Objetivo

Entender como os tópicos de pesquisa estão evoluindo dentro das instituições federais, identificando quais áreas ganham destaque e como a pesquisa prática se alinha com as mudanças tecnológicas atuais.

## 📈 Dados Coletados

- **Período**: 2021-2025
- **Fonte**: SBC OpenLib (SOL)
- **Metodologia**: Análise manual de títulos, resumos e palavras-chave
- **Temas Identificados**: Classificação em 5+ áreas principais de pesquisa

O projeto analisa as seguintes áreas de pesquisa:

## 📝 Metodologia

A coleta de dados foi realizada através de:
1. Acesso à plataforma SBC OpenLib (SOL)
2. Filtragem de edições do ENCompIFs (2021-2025)
3. Leitura sistemática de títulos, resumos e palavras-chave
4. Categorização manual dos trabalhos por similaridade temática
5. Análise e visualização dos padrões identificados
